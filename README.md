# 📚 Learning Projects

Este repositório reúne projetos desenvolvidos durante meus estudos em programação.  
O objetivo é praticar lógica, conceitos de desenvolvimento e aprimorar minhas habilidades através da construção de pequenos sistemas e aplicações.

## 👨‍💻 Sobre mim

Meu nome é **Vinicius Souza**, sou estudante de **Análise e Desenvolvimento de Sistemas** e tenho grande interesse em tecnologia, desenvolvimento de software, cloud computing e dados.

Este repositório faz parte da minha jornada de aprendizado e evolução como desenvolvedor.

## 🚀 Tecnologias utilizadas

As principais tecnologias utilizadas nos projetos deste repositório são:

- Python
- C
- Tkinter
- Turtle
- Pandas
- APIs REST
- Git
- GitHub

## 📂 Projetos

### 🔄 [Conversor](./Conversor)
Aplicação com interface gráfica desenvolvida em **Python com Tkinter** para conversão de unidades de distância.

### 🎮 [Flash Cards de Idiomas](./JogodasPalavras)
Aplicação desenvolvida em **Python com Tkinter** para auxiliar no aprendizado de idiomas utilizando cartões interativos.

### 🗺️ [Jogo de Adivinhação dos Estados dos EUA](./JogodosEstados)
Aplicação interativa feita com **Python e Turtle**, onde o usuário precisa adivinhar os nomes dos estados dos Estados Unidos.

### 📁 [Sistema de Gerenciamento de Livraria](./ManipulaçãoARQ)
Aplicação desenvolvida em **linguagem C** que simula um sistema de gerenciamento de livros utilizando arquivos para armazenamento de dados.

### 🐍 [Jogo da Cobrinha](./jogodaCobrinha)
Implementação do clássico jogo **Snake** utilizando **Python e Turtle**, com arquitetura modular separando a lógica do jogo em diferentes classes.

### 🛰️ [Rastreamento da Estação Espacial Internacional (ISS)](./localSatelite)
Script em **Python** que utiliza **APIs públicas** para obter informações da posição da Estação Espacial Internacional.
